# Purpose
This document defines the visual direction and UI rules for the LensByLey website before final coded implementation. It should be treated as the design system direction source of truth unless replaced by a newer approved revision.

# Design Goals
The LensByLey website should feel:
- editorial
- warm
- premium
- modern
- clean
- image-led
- restrained
- soft
- polished
- trustworthy

The design should feel current to 2026 without relying on trendy gimmicks.

# Color Direction

## Core Palette
The brand should be built around soft neutrals:
- white
- off-white
- cream
- beige
- warm neutrals

## Accent Direction
- subtle only
- muted earthy tones if needed
- no loud accent colors
- no harsh black and white contrast

## General Color Rules
- avoid pure black if possible
- avoid stark clinical white
- keep the palette soft, warm, and premium
- let the photography carry most of the richness

# Typography Direction

## Headlines
- elegant serif
- editorial and refined
- warm and modern
- should feel elevated without feeling pretentious

## Body
- clean neutral sans serif
- highly readable
- restrained and modern

## Typography Rules
- no more than two typefaces total
- headlines should feel calm and confident
- body text should stay clean and practical
- avoid trendy typography experiments
- avoid decorative overload

# Layout Principles
- strong image hierarchy
- generous whitespace
- editorial rhythm
- calm spacing
- sections should breathe
- full-bleed only when earned
- asymmetry can be used carefully to create interest

# Image Treatment Rules
- photography is the design centerpiece
- use large, beautiful imagery
- avoid cluttered multi-image chaos
- keep crops intentional
- make editing consistency a top priority
- avoid mixing too many crop styles without a system
- no carousel-heavy presentation

# UI Behavior
- navigation should be minimal and clean
- buttons should feel refined, not loud
- CTAs should be clear but restrained
- forms should feel polished and easy to use
- cards should be minimal if used at all
- avoid over-framing every section

# Motion Principles
- subtle only
- gentle fades
- soft hover states if needed
- no dramatic reveals
- no parallax
- no cursor effects
- no flashy transitions
- no over-animated sections

# Mobile Principles
- mobile should feel intentionally designed
- typography should scale cleanly
- image crops should be mobile-aware
- spacing should remain breathable
- no desktop layout simply collapsed without thought
- navigation and CTA placement should feel easy and calm

# Components to Keep Minimal
- navbar
- footer
- buttons
- testimonials
- form fields
- service blocks

# Elements to Avoid
Do not introduce:
- clipart icons
- playful event graphics
- package cards
- pricing tables
- checkmark-heavy layouts
- glassmorphism
- gradients
- heavy shadows
- startup/SaaS UI
- visual noise
- cheap template patterns
- over-designed hover effects
- social-media-grid energy

# Homepage Direction Notes
The homepage should feel like:
- a beautiful first impression
- a confident visual introduction
- a curated look into the work
- a soft premium invitation to inquire

The homepage should not feel like:
- a brochure
- a package page
- a wedding-only template
- a generic portfolio site

# Section Rhythm Notes
Recommended homepage flow:
1. Hero
2. About / brand intro
3. Selected work
4. Services preview
5. Testimonials
6. Inquiry CTA
7. Footer

This should stay clean, strong, and minimal.

# Final Reminder
The design should not try too hard to look expensive.
It should feel expensive because it is restrained, consistent, and beautifully curated.
